# Worklogs report

Worklogs report generator


## Setup and usage

* install packages

        yarn install

* prepare fields file

        yarn setup

* edit `fields.json`

* start app

        yarn start

### Docker

* build image

        docker build -t worklogs-report .

* run

        docker run -p 8080:8080 worklogs-report
